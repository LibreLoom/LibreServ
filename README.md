# LibreServ

Self-hosting platform that makes running your own server as simple as plugging in a black box.

## 🚧 Status

In active development - MVP target: April 30, 2026

## 🎯 Goals

- **No terminal required** for 95% of users
- Everything should "just work" out of the box
- Every action should be reversible
- Plain language, not technical jargon

## 🛠️ Tech Stack

- **Backend:** Go 1.22+
- **Frontend:** React + Vite
- **Database:** SQLite
- **Containerization:** Docker
- **Reverse Proxy:** Caddy

## 📦 Features (Planned)

- Three app types: Built-in catalog, Custom uploads, External apps
- Unified monitoring for all apps
- Automatic HTTPS
- Backup system with scheduling
- Automatic updates with rollback
